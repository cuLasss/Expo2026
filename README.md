# Expo2026 - IF Rush

Runner 3D de sessao curta para a banca do IF na exposicao agropecuaria.

## Rodar local

```powershell
python if-rush/server.py --port 8765
```

Abra:

```text
http://localhost:8765/
http://localhost:8765/if-rush/
```
